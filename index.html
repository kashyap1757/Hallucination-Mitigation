🏗️ Architecture

4-Stage Hybrid Pipeline

Each stage maps directly to techniques surveyed in our IEEE conference paper, "Demystifying Machine Imagination."

Stage 1 — §II-A
📚

RAG Retrieval

Retrieval-Augmented Generation grounds the LLM response in factual context before generation begins.

TF-IDF → FAISS
Stage 2 — §II-C, §II-F
🔄

Self-Consistency Sampling

Generate N=5 independent responses and apply majority voting to filter unstable outputs.

N=5 Samples + Vote
Stage 3A — §II-G
📈

Uncertainty Estimation

Compute semantic entropy across samples to quantify the model's internal disagreement.

Semantic Entropy
Stage 3B — §II-F
✅

NLI Verification

Natural Language Inference entailment scoring verifies output against retrieved context.

NLI Entailment
Hallucination Score — Composite Gate
H_score = 0.5 × Entropy(S₁...Sₙ) + 0.5 × (1 − NLI_entailment)
If H_score ≥ threshold → ⚠️ FLAGGED

🚀 Quick Start

Get Running in 60 Seconds

Clone, install, and run the pipeline with just a few commands.

Terminal
# Clone the repository
git clone https://github.com/kashyap1757/Hallucination-Mitigation.git
cd Hallucination-Mitigation

# Install dependencies
pip install -r requirements.txt

# Run the pipeline
python pipeline.py

# Evaluate results
python evaluate.py

# Launch interactive dashboard
streamlit run dashboard.py
